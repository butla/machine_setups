XFCE desktop
============

## Initial setup
- clock widget: set time format to %Y-%m-%d %H:%M:%S
- remove XFCE workspace switcher and set up favourites menu
- restart so that XFCE configuration loads
- battery level widget
- dark theme for the login screen
